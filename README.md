# cse110-sp26-group17

CSE 110 SP2026 — **Team 17**


## Project Description
We’re creating an AI slot machine project where we use a generative AI coding assistant to build the same web-based game multiple times from an identical prompt. Instead of focusing on the final product, we analyze how the outputs vary in quality, features, and correctness. We then refine top results to see how much improvement is possible, helping us understand the strengths and limits of AI as a prototyping tool while working collaboratively as a team.
